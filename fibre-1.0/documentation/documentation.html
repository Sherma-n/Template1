Requirements

For installing this template you need to make sure that you have the following extensions on your webserver. You will need to contact your webhost if the following requirements are missing:

  • PHP 5.2+;
  • Apache webserver (For the Page Builder Windows servers will probably work, but are not supported)
  • PHP Mail support;
  • Enabling JavaScript in the web browser.

To be able to edit and upload the theme you need also:

  • A text editor like Sublime Text 2, Notepad++ or Brackets
  • An FTP tool to upload the files like Filezilla or Cyberduck

Installation

Unzip the downloaded files to the local disk on your computer. Inside the extracted folder there will be a folder with documentation and a folder with the html templates. Choose pages which you will use in project and copy all the content related to the layout that interests you to your server via FTP client.

In this archive some images that you can see in demo pages were replaced by a placeholder.

Structure

As Fibre is an HTML template, it uses a number of .html pages in conjunction with css stylesheets and a number of javascript plugins. The following is a brief overview of the files and a look at some prominent features.

We do our best while making all files for Fibre. Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments – all this for your convenience.

Each demo has similar file structure:

  • css – CSS Styles Folder
  • images – Image Folder
  • js – Javascript Files Folder
  • php – PHP Form Folder
  • index.html – Main HTML File

Html File Structure

This theme is built upon Twitter Bootstrap 3.3 framework and is fully responsive. It adapts to window resolution and looks good on all devices including ipad and iphone. See for yourself (resize your browser).

If you would like to read more about bootstrap framework click here: Bootstrap Documentation

Grid System - Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. Fonts

Fibre uses "Lato" Google WebFont. You can choose other fonts from google directory editing this line of code in the <head>.

<!-- Google Fonts -->
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Lato:300,400,700,900,300italic,400italic,700italic,900italic">

If you want to import new font from Google Webfont library, don't forget to change property in CSS style: font-family with correct font name.

<!-- css/style.css -->

body {
    font-family: "Lato", "Helvetica Neue", Helvetica, Arial, sans-serif;
}

Color Shemes

Check the <head> of the index.html. There you will find all colors with links that are availabe for „Fibre“ theme (comment out). The blue color option is the default color and active. If you want to use an other color than blue e.g. „gold“ please comment in the stylesheet you want.

Dark & Light Version

To adjust the color versions in the whole document, please add the class „dark-version“ for the dark website version or „light-version“ for the light version of the website to your main container. 

<!--- Main -->
<div id="page class="dark-version">
    ...
</div>
<!--- /End Main -->

To define the background colors of the sections, please add one of the following classes to the section you want to change:

  • bg-color: colored background
  • bg-color-light: light colored background
  • bg-color-dark: dark colored background
  • bg-white: white backround
  • bg-light: light backround
  • bg-lightgrey: light grey backround
  • bg-silver: silver backround
  • bg-grey: grey backround
  • bg-darkgrey: dark grey backround
  • bg-dark: dark backround
  • bg-black: black backround
<section id="services class="bg-grey">
    ...
</section>

The colors of the sections / containers are also editable. To change this use one of the following classes:

  • text-color: colored text
  • text-dark: dark grey text
  • text-silver: light grey text
  • text-white: white text
Exception is the customization of the top navigation – click here

Header Navigation

The header navigation needs to be adjusted according to the used color version of the template respectively the used Hero section. For this use the following classes:

  • light-nav: use this for a light website version and a light hero section
    Your logo should be the logo-blue.png (if you use an additional stylesheet for colors than use logo-(yourcolor).png
  • dark-nav: use this for a dark website version and grey hero section
    For the dark version and grey hero section (like in the dark slider version) your logo should be the logo-silver.png
  • dark-nav-colored-hero: use this for a dark website version and dark hero section
    For this version please use the logo-silver.png and the logo-white.png 
<header id="nav class="light-nav">
    ...
</header>

Smooth Scroll

For smooth-scroll you need to add the class „smooth-scroll“ to the a-tag. The hyperlink refers to the corresponding id of the relative section. 

<a href="#section" class="smooth-scroll">Link</a>

<section id="section">
    ...
</section>

Back to Top Button

The "Back to Top" link allows users to smoothly scroll back to the top of the page. It's a little detail which enhances navigation experience on website with long pages. You can disable it on your page by uncommenting in your index.html these lines of code at the end of the file:

<a href="#page" class="btn-circle btn-circle-lg btn-color btn-top smooth-scroll">
    <i class="fa fa-angle-up"></i>
</a>

Animation Effects

Fibre uses ScrollReveal-Plugin to trigger animations on scroll. To learn more about ScrollReveal please read the Documentation of the Plugin.

Predefined initialization in the custom.js

<!—- HTML -—>
<div class="reveal-bottom-opacity">
    ...
</div>

// Javascript
window.sr = ScrollReveal();
sr.reveal('.reveal-bottom-opacity', {
    origin: 'bottom',
    distance: '20px',
    duration: 800,
    delay: 0,
    opacity: 0,
    scale: 0,
    easing: 'linear'
});

Youtube / Vimeo Embed

Embedding videos of Youtube or Vimeo (like the example in the fibre video section) will be accomplished through "data-url" attribute. The video will be embedded as recently as a button event has happen to the video container. As long as there was no button presses by viewer there will be shown an thumbnail of the video. 

<!—- HTML -—>
<div id="video-embed" class="video-wrapper" data-url="https://vimeo.com/117570710">
    ...
</div>

// Javascript
Video('#video-embed');

Charts

The Fibre Template uses the Circle Plugin to show animated pie charts. The charts could be defined through the attributes data-value, date-unit und data-percentage.

  • data-value: responsible for text/numbers in the chart
  • date-unit: Unit (subscript) in the chart *
  • data-percentage: Value in percent (here: colored circle (50 = 180°)

Hero Section

Solid Background

To change the background image in the header you need to go to the style.css, scroll down to „03. Backrounds“ and change the colors after „#“ to the colors of your needs.

Image Background

To change the background image in the header you need to go to the style.css, scroll down to „10. Hero Section“ and change the link of the image to the name you stated for your header image. Please make that you copy your image into the images folder.

You can also open the image „hero-image-bg.jpg“ in Photoshop

  • place your new image
  • Press „Save for Web“
  • and save the new image into the images folder of the fibre website and overwrite the old image file.

Youtube Background

Fibre provides the opportunity to embed a youtube-video in background.

Youtube video embedding accomplished through the attribute data-video-id. You only need to provide the video ID which you can find in the url of the video:

https://www.youtube.com/watch?v=_M1A_nQZw0Q

Youtube video is only enabled for Desktop – videos for mobile and tablets are disabled. Only fullscreen poster will be used.

Use the img-tag to change the name of the preview image of the video.

<div id="youtube-background" class="hero-video" data-video-id="_M1A_nQZw0Q">
    <img src="images/hero-youtube-bg.jpg" alt="">
</div>

Video Background

Fibre provides the opportunity to add a video in background(which should be composed of a screenshot jpg, a file .mp4, a file .ogv and a file .webm).

To change the file name and the directory where the video is please use the attributes data-path und data-file. 

<div id="video-background" class="hero-video" data-file="video">
    <img src="images/hero-video-bg.jpg" alt="">
</div>
Video is only enabled for Desktop - videos for mobile and tablets are disabled. Only fullscreen poster will be used. Use the img-tag to change the name of the preview image of the video.

Forms

Newsletter Form

Current active subscribe form will save emails to your mailchimp list. If you don't have a Mailchimp account you alternative could use the simple php form above:

Write your email and subject prefix in newsletter-process.php:

and use:

<!-- HTML -->
<form action="php/newsletter-process.php" method="post" id="newsletter-form" class="wrapper-h-xs-md">

<!-- instead of
<form id="mc-form"> -->

In order to use the respective Mailchimp subscription form you have to replace the mailchimp post url in the url attribute in custom.js.

// Javascript
$('#mc-form').ajaxChimp({
  ...

  // Replace the URL above with your mailchimp URL (put your URL inside '').
  url: 'http://please enter your valid url to mailchimp'
});
How to find the mailchimp post url?
  1. Login to your mailchimp account
  2. Go to your list
  3. Select Signup Forms
  4. Select Embedded forms
  5. Copy the link and paste into action attribute. (see image below)

Contact Form

To connect the contact form to your email account please enter your email address into the contact-process.php under: $emailTo = ' ';

Additionally you are able to change the subject line of the email you will get after sending a message for this please use the php line: $subjectPrefix = ' ';

To make bigger changes to the included forms you will need professional skills in PHP and Javascript. Please be careful.

Further adjustments (e.g. adding an additional form field) can be made in the HTML form, contact-form.js und contact-process.php
